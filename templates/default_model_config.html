4 使用说明

默认 model id 可直接用于 bedrock-runtime 的 converse / invoke_model API，无需提前创建 inference profile。


适用场景：直接测试模型、低并发调用、不需要跨区域路由。

inference profile 优势：跨区域自动路由、更高配额、支持企业标签管理。
